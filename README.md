# ElitePhoto Studio - Professional Photography Studio Management System

A complete Flask-based web application for a professional photo studio with gallery management, booking system, and admin dashboard.

## Features

- 🎨 **Professional & Creative Design** - Modern, responsive UI with smooth animations
- 📸 **Photo Gallery** - Upload, categorize, and display photos with thumbnails
- 📅 **Booking System** - Client booking management with session types
- 📊 **Admin Dashboard** - Statistics, recent bookings, and photo management
- 💬 **Contact System** - Client inquiry management
- 🔒 **Secure Upload** - File validation and thumbnail generation
- 📱 **Fully Responsive** - Works on all devices
